# nix-meshcore

This repository contains (non-functional) configurations for:

- my (2) Heltec v3 running MeshCore MQTT Observer firmware and
- my Seeed Studio SenseCAP Solar Node P1-Pro running stock MeshCore firmware

## Heltec v3 MQTT Observers

Flash the [Observer firmware](https://observer.gessaman.com/) and follow your regional guide for MQTT.

Once you're acquainted with MeshCore, you're free to use this project as reference for configuring your node.

## Seeed Studio SenseCAP Solar Node P1-Pro

Flash the [official MeshCore firmware](https://meshcore.io/flasher) and follow your regional guide for configuration.

Once you're acquainted with MeshCore, you're free to use this project as reference for configuring your node.
